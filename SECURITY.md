# Security Policy

## Supported Versions

This project generally supports the latest released version on the default branch.

If you believe you have found a security vulnerability, please **do not** open a public issue.

## Reporting a Vulnerability

- Please report security issues privately.
- If GitHub "Private vulnerability reporting" is enabled for this repository, use that feature.
- Otherwise, email the maintainer: Marouan-chak.

### What to include

- A description of the issue and impact
- Steps to reproduce
- Affected versions/commits
- Any proof-of-concept (redact secrets)

## Disclosure

We will acknowledge receipt, investigate, and provide an update as soon as reasonably possible.
